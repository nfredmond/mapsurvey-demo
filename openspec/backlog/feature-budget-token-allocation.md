# Budget/token allocation question type

**Type**: feature
**Priority**: medium
**Area**: frontend
**Created**: 2026-04-02

## Description

New question type where respondents distribute a fixed number of tokens or virtual dollars across a set of categories (e.g., "Parks", "Housing", "Roads"). The UI enforces a total budget constraint — respondents cannot exceed the limit. Key for participatory budgeting and "Invest in Change" engagement events used in master plan workshops.

## Notes

- See [UC-01: Token Voting at Engagement Event](../docs/use-cases/uc01-token-voting.md)
- UI options: sliders, +/- buttons, or drag-and-drop allocation
- Each category should support an optional description and icon/image
- Results aggregate naturally into bar/pie charts for the analytics dashboard
