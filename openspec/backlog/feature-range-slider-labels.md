# Range slider: from-to labels

**Type**: feature
**Priority**: high
**Area**: frontend
**Created**: 2026-03-26

## Description

Add descriptive labels below the range slider showing the "from" and "to" text. Currently the slider has no visible indication of what the endpoints mean. Users define choice names for range values but they aren't displayed as endpoint labels.

## Notes

- Source: Manuel Frost (manu04) — his survey uses 9-point scales like "(positive) Geräusche" to "(negativer) Lärm"
- Quick fix — display first and last choice names as labels below the slider
