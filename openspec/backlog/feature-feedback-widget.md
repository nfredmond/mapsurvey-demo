# Feedback Widget

**Type**: feature
**Priority**: medium
**Area**: general
**Created**: 2026-03-30

## Description

Плавающий виджет обратной связи в стиле Intercom/Crisp — кнопка в правом нижнем углу, открывающая pop-up с формой "Send us a message". Должен работать без регистрации и авторизации, т.к. вопросы могут касаться именно процесса входа/регистрации. Виджет доступен на всех страницах (публичные опросы, landing, editor).

## Notes

Варианты реализации:
- **Внешний сервис (рекомендуется для старта)**: Crisp, Tawk.to (бесплатный), Intercom — подключение через JS-сниппет в base-шаблоне, минимум кода
- **Собственная реализация**: модель `Feedback(email, message, page_url, created_at)` + floating button + pop-up форма + отправка на email админа
- Можно начать с внешнего виджета (Crisp/Tawk.to) и позже мигрировать на собственное решение

Ключевые требования:
- Без авторизации — анонимная отправка (опционально: поле email)
- Плавающая кнопка в правом нижнем углу на всех страницах
- Минимальная форма: email (опционально), сообщение, отправка
