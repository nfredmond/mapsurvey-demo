# Answer choice templates

**Type**: feature
**Priority**: low
**Area**: frontend
**Created**: 2026-03-26

## Description

Pre-built templates for common answer choice sets: 5-point Likert scale, Yes/No/Maybe, age ranges, frequency scales, etc. Users can select a template instead of typing choices manually.
