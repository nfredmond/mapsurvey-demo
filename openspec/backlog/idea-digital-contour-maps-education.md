# Digital Contour Maps — Geography Education Use Case

**Type**: idea
**Priority**: medium
**Area**: general
**Created**: 2026-03-26

## Description

A geography teacher (lrbenedict12) is using Mapsurvey as interactive digital contour maps: students draw regions (e.g. "Draw a bounding box around the Midwest") on a real map instead of paper. 28 responses per session = full classroom. Monthly recurring sessions (March/April editions). This is a validated, recurring use case with a huge potential audience (every geography teacher worldwide).

## Needs (Product)

- **Survey duplication** — easy cloning for next month's session
- **Answer overlay map** — display all student responses on one map (to show "here's how everyone drew the Midwest")
- **Teacher dashboard** — view/compare student submissions side by side
- **Survey templates** — pre-built "contour map" template teachers can fork

## Needs (Marketing)

- Landing page targeting educators ("Replace paper contour maps with interactive ones")
- Example survey template that teachers can try immediately
- Outreach to geography teacher communities (Reddit r/geography, teacher forums, education conferences)
- Case study / showcase based on lrbenedict12's usage (with permission)
