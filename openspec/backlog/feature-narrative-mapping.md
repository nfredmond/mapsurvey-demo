# Narrative mapping (story + map point)

**Type**: feature
**Priority**: low
**Area**: frontend
**Created**: 2026-04-02

## Description

A composite question type combining a map point with a story title, text summary, and category tag. Supports repeatable entries within one survey session — an interviewer can record multiple location-tied stories from a single interviewee. Based on the "Chat Maps" / Narrative Infrastructure method for community engagement.

## Notes

- See [UC-05: Narrative Mapping Interview](../docs/use-cases/uc05-narrative-mapping.md)
- Requires repeatable question groups (new concept for Mapsurvey)
- Story map view: pins with popups showing title + text + category
- Category filtering on map
- Optional consent checkbox per story ("OK to share publicly?")
- Future: audio recording per story (relates to backlog #36 audio upload)
