# WMS/WFS basemap support

**Type**: feature
**Priority**: medium
**Area**: frontend
**Created**: 2026-03-26

## Description

Allow survey creators to configure custom basemap layers via WMS or WFS URLs. This enables overlaying institutional geodata (e.g. Berlin's GDI, national mapping services) on the survey map.

## Notes

- Source: Manuel Frost (manu04) — "great and important!" for Berlin Senate workflows
- Leaflet supports WMS/WFS via L.tileLayer.wms — needs UI in survey editor to configure per-survey
