# Satellite and topographic basemap options

**Type**: feature
**Priority**: medium
**Area**: frontend
**Created**: 2026-03-26

## Description

Add satellite imagery and topographic map options as basemap choices for surveys. Currently only the default OpenStreetMap tiles are available. Nature/ecology and rural use cases especially benefit from satellite view.

## Notes

- Source: Marijana Jericevic (Galanthus) — citizen science snowdrop mapping in rural Brittany
- Also requested by Manuel Frost (manu04)
- Options: Esri World Imagery (free for non-commercial), Mapbox Satellite, OpenTopoMap
- Could be per-survey setting in editor
