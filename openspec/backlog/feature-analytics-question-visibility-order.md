# Analytics: hide/show and reorder questions

**Type**: feature
**Priority**: medium
**Area**: frontend
**Created**: 2026-04-03

## Description

As an analyst, I want to hide certain questions from the analytics dashboard while keeping the ability to bring them back later. I also want to reorder how questions are displayed in the dashboard, independently from the survey structure order.

## Notes

- Hide/show should be per-survey, not per-question-type
- Reorder should persist across sessions (likely stored server-side)
- Hidden questions should be easily discoverable (e.g. a "show hidden" toggle or a settings panel)
