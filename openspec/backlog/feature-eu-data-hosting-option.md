# EU Data Hosting Option

**Type**: feature
**Priority**: medium
**Area**: infra
**Created**: 2026-04-05

## Description

Offer users a choice between global and EU-based server during signup or organization setup, similar to KoboToolbox's server selection (Global vs European Union). The EU server would be hosted within the European Union to satisfy GDPR and data residency requirements for organizations that need or prefer EU-based data storage.

## Notes

- Reference: KoboToolbox offers Global + EU (Ireland) server selection on their signup page
- Related to #10 (Frankfurt server migration) — that item covers the infra move; this covers the user-facing multi-region selection UX
- Consider whether this is a signup-time choice, an organization setting, or per-survey
