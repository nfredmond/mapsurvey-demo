This is Artem from Mapsurvey. Today I shipped something I've been working on for a while — the Analytics Dashboard. It's in beta, and you can try it right now.

What you can do:

  • See responses on a map — points, lines, polygons, all at once. Add heatmap layers to find patterns.
  • Browse data in a table — sort, filter, search, expand any session to see details with a mini-map.
  • View charts — choice distributions, numeric ranges, time trends. Everything cross-filters with the map and table.
  • Track performance — completion funnel, time per section, device and referrer breakdown, UTM campaigns.
  • Flag bad data — automatic detection of fast completions and duplicate submissions.
  • Split your screen — VS Code-style panes. Map + table, charts + map — whatever works for you.

To try it: open any survey in the editor → click Analytics in the sidebar.
